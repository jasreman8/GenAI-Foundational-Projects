# GenAI-Foundational-Projects
This repo contains all the GenAI foundational projects.

**Notebook 1:** A beginner-friendly notebook demonstrating how to authenticate, call OpenAI APIs, and structure prompts to analyze and extract intelligence from clinical narrative text.

**Notebook 2:** A companion notebook to the OpenAI implementation, this project demonstrates how to run medical text extraction using Llama3 models deployed locally through Ollama, with a focus on understanding prompt formatting and API request structure.
